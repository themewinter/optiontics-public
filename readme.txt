== Description ==

[**Optiontics**](https://themewinter.com/optiontics/) is an advanced **WooCommerce product add-ons** and **custom option manager** plugin that gives you the freedom to sell personalized or configurable products effortlessly.

Whether you're selling pizzas with multiple toppings, shirts with custom prints, or tech gadgets with extra accessories — Optiontics provides a **visual form builder**, **dynamic pricing**, and **seamless integration** with WooCommerce checkout.

== 🧩 Key Features ==

### 🎨 Visual Add-on Form Builder
- Drag-and-drop builder for creating custom product forms.
- Add fields like **Text, Number, Select, Checkbox, Radio, Color, File Upload**, and more.
- Display total price dynamically using real-time calculations.

### 💰 Dynamic Pricing Rules
- Add fixed or percentage-based price for each add-on option.
- Combine base product price with add-on price in the cart total.
- Supports **tiered pricing**, **conditional pricing**, and **quantity-based price logic**.

### 🧾Smart Option Management
- Group add-ons into reusable templates and assign them to multiple products.
- Manage global or product-specific add-ons from one dashboard.
- Enable/disable specific fields based on category, tag, or variation.

### ⚙️ Product Variation Integration
- Combine WooCommerce product variations with add-on options.
- Add “Extra toppings,” “Extended warranty,” “Custom engraving,” etc. to product variations seamlessly.

### 🔔 Real-time Price Display
- Instantly update product total as users select or change add-on options.
- Show breakdown of add-on costs for full transparency.

### 📦 Checkout & Order Sync
- Store all add-on data in WooCommerce order meta.
- Display add-on selections in order emails, admin order view, and customer dashboard.

### 🔐 Validation & Security
- Validate user input (e.g., required fields, max length, file type).
- Securely sanitize and store custom field values in WordPress database.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/optiontics/`, or install the plugin through the WordPress plugin installer.
2. Activate the plugin through the “Plugins” menu in WordPress.
